# Table — متتبع العادات — Vue 3 + Vite (العربية)

تطبيق متعدد الوظائف لتتبع العادات مع دعم المهام، المهام الفرعية، العلامات اليومية، تصدير/استيراد البيانات، وتغيير السمة واللغة. التقنيات: Vue 3، Vite، Pinia، TailwindCSS، i18n، ESLint، Prettier.

---

## دعم اللغات
- اللغات المدعومة: الروسية، الإنجليزية، العربية، الإسبانية، الصينية
- جميع نصوص الواجهة مترجمة (انظر `src/locales/`)
- يمكنك تغيير لغة التطبيق من الإعدادات

## الميزات
- إضافة، تعديل، حذف المهام والمهام الفرعية
- علامات يومية لكل مهمة/فرعية
- التنقل بين الأشهر، عرض الأرشيف
- تصدير واستيراد البيانات (JSON)
- تغيير السمة (فاتح/داكن/النظام)
- تغيير اللغة (الروسية/الإنجليزية/العربية/الإسبانية/الصينية/النظام)
- سحب وإفلات للمهام (vue-draggable-next)
- تصميم متجاوب، دعم الطباعة
- إعدادات عرض الأسابيع: فواصل الأسابيع، اختيار بداية الأسبوع (الاثنين/الأحد)
- اختصارات لوحة المفاتيح للتحكم السريع
- جميع الإعدادات محفوظة بين الجلسات

## هيكل المشروع
- `src/App.vue` — المكون الرئيسي للتطبيق
- `src/main.js` — تهيئة Vue و Pinia و i18n
- `src/stores/habitStore.js` — Pinia store: منطق المهام، التقدم، السمة، اللغة، التصدير/الاستيراد، إعدادات الأسابيع
- `src/locales/` — ملفات الترجمة
- `src/components/` — مكونات التطبيق
- `src/style.css` — TailwindCSS والأنماط العامة
- الإعدادات: `vite.config.js`، `eslint.config.js`، `.prettierrc`

## البدء السريع
```sh
pnpm install         # تثبيت الاعتمادات
pnpm run dev         # بدء خادم التطوير
pnpm run build       # بناء للإنتاج
pnpm run lint        # تدقيق الكود
pnpm run format      # تنسيق الكود
```

## الاعتمادات الرئيسية
- [Vue 3](https://vuejs.org/) — إطار العمل
- [Vite](https://vitejs.dev/) — أداة البناء
- [Pinia](https://pinia.vuejs.org/) — إدارة الحالة
- [TailwindCSS](https://tailwindcss.com/) — الأنماط
- [vue-i18n](https://vue-i18n.intlify.dev/) — الترجمة
- [vue-draggable-next](https://github.com/anish2690/vue-draggable-next) — السحب والإفلات
- [lucide-vue-next](https://lucide.dev/) — الأيقونات

## الهندسة والمنطق
- جميع بيانات المهام والعلامات مخزنة في localStorage (أرشيف حسب الشهر)
- Pinia store (`habitStore.js`) يدير المهام، المهام الفرعية، التقدم، السمة، اللغة، التصدير/الاستيراد، إعدادات الأسابيع
- الترجمة عبر vue-i18n، الترجمات في `src/locales/`
- السمة، اللغة، إعدادات الأسابيع محفوظة
- السحب والإفلات عبر vue-draggable-next

## اختصارات لوحة المفاتيح
- Escape — إعادة تعيين الأوضاع، إغلاق القائمة/الإعدادات
- E — تبديل وضع الممحاة
- D — تبديل وضع الحذف
- N — إضافة مهمة جديدة
- السهم الأيسر/الأيمن — تبديل الشهر
- S — فتح/إغلاق الإعدادات

## المساهمة
- اتبع نمط الكود (Prettier, ESLint)
- استخدم المكونات و store الموجودة
- للغات جديدة — أضف الترجمات في `src/locales/`
- قبل طلب الدمج: `pnpm run lint && pnpm run format`
